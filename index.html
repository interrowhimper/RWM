ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ Root Word Method Drill

Welcome to the Root Word Method drill! This drill is intended to help you work on conjugating verbs and, optionally, adding affixes.
You're responsible for checking your own work! Please rely on materials generated by first-language speakers.
As you're working, don't forget your root suffixes, and be sure to consider whether your verb needs a Set A or Set B pronoun.
This drill is quite open-ended -- don't be discouraged! A great way to use it is to try and write a Cherokee sentence around the word
that is randomly generated for you.
